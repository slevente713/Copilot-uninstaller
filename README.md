# Copilot-uninstaller
This .ps1 script is used to uninstall copilot from Windows 11/10 but from everywhere; from registry, from the applications, and from services. Ez a .ps1 szkript a Copilot eltávolítására szolgál a Windows 11/10 rendszereken, de mindenhol eltávolítja; a Regedit-ből, az alkalmazások közül és a szolgáltatások közül is. Dieses .ps1-Skript dient zur Deinstallation von Copilot auf Windows 11/10-Systemen und entfernt das Programm vollständig – sowohl aus der Registrierungsdatenbank als auch aus der Liste der Anwendungen und Dienste. この.ps1スクリプトは、Windows 11/10システムからCopilotをアンインストールするためのものです。レジストリ、アプリ、サービスなど、あらゆる場所から完全に削除します。Этот скрипт .ps1 предназначен для удаления Copilot в системах Windows 11/10; он удаляет программу полностью: из Реестра, из списка приложений и из списка служб. 

How to use? Download the Copilot uninstaller.ps1 file. Right-click on the downloaded .ps1 file, and select: Run with Powershell. IMPORTANT NOTE: Your screen may go completely black for a few seconds because explorer.exe restarts. ВАЖНОЕ ПРИМЕЧАНИЕ: Экран может на несколько секунд полностью потемнеть из-за перезапуска процесса explorer.exe. 【重要】explorer.exe が再起動するため、画面が数秒間完全に真っ暗になる場合があります。WICHTIGER HINWEIS: Der Bildschirm kann für einige Sekunden komplett schwarz werden, da explorer.exe neu gestartet wird. FONTOS MEGJEGYZÉS: Előfordulhat, hogy a képernyő néhány másodpercre teljesen elsötétül, mivel az explorer.exe újraindul.
