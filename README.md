# Copilot-uninstaller
Ez a .ps1 szkript a Copilot eltávolítására szolgál a Windows 11/10 rendszereken, de mindenhol eltávolítja; a Regedit-ből, az alkalmazások közül és a szolgáltatások közül is. This .ps1 script is used to uninstall copilot from Windows 11/10 but from everywhere; from registry, from the applications, and from services. 

Hogyan használd? Rámész a Debloater Lite fájlra, és kimásolod a kódot. Megnyitod a Powershell-t Admin jogosultsággal, és beilleszted a kódot. Megvárod hogy lefusson, és utánna indísd újra a géped. FONTOS MEGJEGYZÉS: Előfordulhat, hogy a képernyő néhány másodpercre teljesen elsötétül, mivel az explorer.exe újraindul. How do you use it? Click on the Debloater Lite file and copy the code. Open PowerShell with administrator privileges and paste the code. Wait for it to run, then restart your computer. IMPORTANT NOTE: Your screen may go completely black for a few seconds because explorer.exe restarts.
